# Text-to-Video Generator

## Overview
Text-to-Video Generator is an AI-powered application that converts text prompts into short videos. The system generates images, narration, and combines them into a final video.

## Features
- Convert text prompts into videos
- AI-generated images
- Text-to-Speech narration
- Background music support
- Video download option
- User-friendly interface

## Tech Stack

### Frontend
- React.js

### Backend
- FastAPI (Python)

### AI Models
- Hugging Face Transformers
- Diffusers
- Text-to-Speech Models
